Detailed Description

Class to represent and manipulate Arnold extensions.

This class is used in the initializeExtensions and deinitializeExtension functions of a MtoA extension to respectively register and deregister the extension services (nodes, translators) with MtoA and Maya.RT CExtension

const CPxArnoldNode &  arnoldNode

Register a Maya node for the given Arnold node.

Certain optional node-level metadata can be used to control how the node factory processes the node's registration:

  1. "maya.name" - the name for the generated or corresponding maya node
  2. "maya.id" - the maya node id to use for the generated or corresponding maya node
  3. "maya.class" - classification string (defaults to "shader/surface")

See CBaseAttrHelper for parameter-level metadata for controlling attribute creation

Parameters:
arnoldNodeEntryarnold AtNodeEntry from which to generate the new Maya node
Returns:
MStatus::kSuccess if the node is registered successfully, else MStatus::kFailure

Register a translator for the given Arnold node entry.

Certain optional node-level metadata can be used to control how the node factory processes the node's registration:

  1. "maya.name" - the name for the generated or corresponding maya node
  2. "maya.id" - the maya node id to use for the generated or corresponding maya node
  3. "maya.translator" - the name to use for the translator If no creator function is provided, will use default translator for each Arnold exported node types (camera, light, shader, shape)
Parameters:
arnoldNodeEntryArnold node name the translator should produce
Returns:
